Мои индивидуальные правки включают в себя только улучшение моего модуля data_loader.py и корректировку main.py с учетом новых выводов.
Комментарии в коде подробно раскрывают функционал и структуру, например, процесса создания диаграмм.
Сначала я добавила в анализ имеющиеся в датасете данные о предыдущих баллах и занятиях с репетитором, включив их в список student, рассчитав для них корреляцию и добавив в отчет.
Затем я создала диаграммы рассеивания, сетку 2х3, сохранив ее в scatter_plots.png. В ней показаны 5 диаграмм рассеивания для зависимостей итогового балла и изучаемых показателей и 1 диаграмма для "независимых" факторов.
Это помогает пользователю оценивать корреляцию не только для текущего балла, но и для прошлого.
Кроме того, я добавила аналитику для пар факторов. Это позволяет фокусироваться не на единичных значениях, а рассматривать факторы как стиль жизни, влияющий на учебу и баллы.
Все новые значения добавила в вывод в конце кода.
Результаты отчета добавлены в текстовый файл analysis_report.txt
